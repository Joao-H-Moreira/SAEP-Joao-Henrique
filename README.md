Instale o Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

Siga os passos

```sh

git clone <YOUR_GIT_URL>


cd <YOUR_PROJECT_NAME>


npm i


npm run dev
```
